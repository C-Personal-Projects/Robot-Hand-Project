# Robot Hand Project
This robot hand was designed and manufactured in order to compete in a competition for the American Society of Mechanical Engineers at the University of Central Florida. The hand achieved first place in the competition. It was desigend so that it may express several degrees of motion, allowing it to maintain a superior grip.
